## v7.1 核心機制 (純連續)

### 1. 日內逆境 (數據驅動擬合)
```python
day_factor = 1 + k_day × hours^n_day
# k_day = 2.29e-5, n_day = 6.6 (優化器擬合)
# 6h:  factor = 4.1
# 12h: factor = 304.7
# 12h/6h = 73.8
```

### 2. 夜間節律 (連續冪函數)
```python
circ = 1 + k_night × hours_in_dark^n_night
# k_night = 0.15, n_night = 2
```

### 3. LAI 脆弱性 (Sigmoid 型)
```python
vuln = cap × (LAI_ref/LAI)^n / (cap + (LAI_ref/LAI)^n)
```

### 4. Stress 非線性 (Michaelis-Menten 型)
```python
nl = 1 + k × Stress / (K + Stress)
```

## ROS 動力學探索總結

### 核心發現

1. **n ≈ 7 無法從單一 ROS 反應動力學推導**
   - 反應動力學最高約 3-4 級
   - 但多層級聯效應可以產生更高的有效冪次

2. **`simulate_uva_model_ros.py` 中的 ROS 狀態變量沒有實際貢獻**
   - ROS 被 APX/CAT 快速清除，維持在接近 0
   - 真正起作用的是 `depletion_factor`

3. **解決方案：數據驅動擬合**
   - 讓優化器自己找出最佳 n_day
   - 優化器收斂到 n ≈ 6.6
   - 證明這是數據要求的，不是人為預設

### 建議

- **使用 v7.1 模型**（數據驅動擬合版）
- 論文中強調「參數由數據自動擬合」
- 用 ROS 級聯效應 **解釋** 高冪次的生物學意義
