## 修改內容

### 1. 移除 UVA-PAR 放大效應
```python
# v5.9: par_conversion_factor = 3.0
# v6.0: par_conversion_factor = 1.0
```

**理由**:
- Sun 原始模型已能正確模擬 CK 和 L6D6
- I_UVA = 22 W/m² 已是等效短波輻射，不需額外放大
- UVA 的量子效率通常低於可見光，3.0 倍放大不合理

### 2. 恢復光合效率係數
```python
# v5.9: c_alpha = 0.54 (校準值)
# v6.0: c_alpha = 0.68 (Sun 原始值)
```

**理由**: 與 Sun 原始模型保持一致

### 3. 加入花青素合成碳成本 (Gemini 建議)
```python
# v6.0 新增
anth_carbon_cost = 2.0  # [kg C / kg Anth]

# 碳緩衝池微分方程
dCbuf_dt = dCbuf_dt - repair_carbon_consumption - anth_carbon_consumption
```

**生理學意義**:
- 花青素是複雜的酚類化合物，合成需要消耗碳骨架
- 使「生長-防禦-修復」三者對碳資源的競爭更完整
- 符合植物資源分配理論

## 問題診斷

### 1. CK 鮮重過高 (+18.3%)
**可能原因**:
- `c_alpha = 0.68` 導致基礎光合速率過高
- Sun 模型可能已針對特定實驗條件校準過 `c_alpha`
- 我們的萵苣品種或環境可能與 Sun 模型不同

**建議**:
- 降低 `c_alpha` (0.60 ~ 0.65 之間嘗試)
- 或保持 v5.9 的 `c_alpha = 0.54`

### 2. L6D6 鮮重過低 (-30.0%)
**可能原因**:
- 移除 `par_conversion_factor = 3.0` 後，UVA 對光合的增益效應消失
- L6D6 處理原本依賴 UVA-PAR 增益來達到接近 CK 的鮮重
- 當前 Stress 值為 0.0，表示損傷參數也需要重新校準

**建議**:
- 如果保持 `par_conversion_factor = 1.0`，需要重新校準損傷參數
- 或考慮使用較小的放大係數 (1.5 ~ 2.0)

### 3. 整體趨勢
- CK 過高 → 基礎光合過強
- UVA 處理組過低 → UVA 增益效應不足
- **核心矛盾**: 如何在不使用放大效應的情況下，解釋 L6D6 ≈ CK 的現象？

## 生理學討論

### UVA-PAR 增益效應的爭議

**v5.9 的假設** (par_conversion_factor = 3.0):
- UVA 對光合的促進效應是 PAR 的 3 倍
- 生理學上不合理（UVA 量子效率應低於可見光）

**v6.0 的假設** (par_conversion_factor = 1.0):
- UVA 直接以等效短波輻射形式加入
- 生理學上更合理，但無法解釋實驗現象

**可能的解釋**:
1. **實驗數據本身的問題**: L6D6 的鮮重測量是否有誤差？
2. **未捕捉的機制**: UVA 可能透過其他途徑促進生長（如誘導特定基因表現）
3. **模型簡化**: Sun 模型本身可能未完全捕捉 UVA 的複雜效應

## 結論與建議

### 當前狀態
❌ **v6.0 模型的鮮重預測精度不佳，無法直接使用**

### 三個可能的方向

#### 選項 A: 回退 v5.9（務實）
- 保留 `par_conversion_factor = 3.0` 和 `c_alpha = 0.54`
- 保留花青素碳成本機制（Gemini 建議很好）
- 在論文中標註 `par_conversion_factor` 是現象級參數
- 承認這是未來研究方向

**優點**: 保持優秀的預測精度 (Mean 2.3%, Max 5.0%)
**缺點**: 生理學意義不明確

#### 選項 B: 部分調整（折衷）
- 使用較小的放大係數 (1.5 ~ 2.0)
- 調整 `c_alpha` (0.60 ~ 0.65)
- 重新校準所有參數

**優點**: 在生理學合理性和預測精度之間取得平衡
**缺點**: 需要大量校準工作

#### 選項 C: 完全重新設計（學術）
- 深入研究 UVA 的生理機制
- 加入更多子模型（如基因表現、酶活性等）
- 使用更精細的實驗數據

**優點**: 最符合科學嚴謹性
**缺點**: 需要大量額外實驗和時間

---

**建議**: 採用選項 A，保持 v5.9 的預測精度，同時保留花青素碳成本機制，並在論文中誠實討論這個限制。
