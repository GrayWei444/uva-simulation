# L6D6 問題解決方案

**日期**: 2025-12-22
**問題**: L6D6 應該是最高鮮重（有額外 22 W/m² 輻射），但預測卻比 CK 低 18.3%

---

## 🔍 問題診斷

### 實驗觀察 vs 模型預測

| 指標 | CK | L6D6 | 差異 |
|------|-----|------|------|
| **實驗鮮重** | 87.0g | **91.4g** | **+5.1%** ✅ UVA促進生長 |
| **模型預測** (調整前) | 87.8g | 74.7g | **-14.9%** ❌ 預測抑制生長 |

**矛盾**: 實驗顯示 L6D6 促進生長，模型預測抑制生長！

---

## 🎯 根本原因

### L6D6 配置

- UVA 強度: **22 W/m²**
- 照射時間: **10:00-16:00** (6 小時/天)
- 照射天數: **Day 29-35** (6 天)

### 日累積能量計算

```
E_daily = I_UVA × hours × 3.6
        = 22 W/m² × 6 h × 3.6 kJ/Wh
        = 475.2 kJ/m²
```

### 原始參數問題

**原始值**:
- `E_50 = 237.6 kJ/m²`  (≈ 11 W/m² × 6h)
- `E_scale = 39.6 kJ/m²`

**正規化能量**:
```
normalized_E = (E_daily - E_50) / E_scale
             = (475.2 - 237.6) / 39.6
             = +6.0  ← 遠大於 0！
```

**觸發非線性放大**:
```
softplus(+6.0, 3.0) ≈ 6.0
intraday_factor = 1 + 1.5 × 6.0² = 1 + 54 = 55x ❌
```

**結果**: L6D6 的損傷被放大 **55倍**！

---

## ✅ 解決方案

### 調整日內非線性參數

**問題分析**:
- 原始 `E_50 = 237.6 kJ/m²` 是針對 **11 W/m² 照射 6 小時** 設計
- 但 L6D6 是 **22 W/m²**，能量翻倍
- L6D6 的日累積能量 (475.2 kJ/m²) 遠超 E_50，觸發了非線性放大
- 但 **L6D6 只照射 6 小時/天**，理論上不應該觸發非線性（修復系統有足夠時間恢復）

**修正策略**:
- 提高 `E_50`，讓 **短時照射 (6h)** 不觸發非線性
- 保持長時照射 (12h) 仍會觸發

### 新參數值

```python
# 舊值
E_50 = 237.6 kJ/m²  # ≈ 11 W/m² × 6h
E_scale = 39.6 kJ/m²

# 新值
E_50 = 600.0 kJ/m²  # v6.0: 提高讓短時照射不觸發非線性
E_scale = 100.0 kJ/m²  # 同比例調整
```

## 💡 生理學理由

### 為什麼 6h 照射不應該觸發非線性？

1. **植物有即時修復能力**
   - 光修復 (photorepair) 在照射期間就進行
   - 暗修復 (dark repair) 在夜間 18 小時充分進行

2. **6 小時照射 + 18 小時夜間**
   - 修復時間 (18h) >> 照射時間 (6h)
   - 修復系統有足夠時間恢復

3. **12 小時照射 + 12 小時夜間**
   - 修復時間減少
   - 修復系統可能飽和
   - 應該觸發非線性

### 為什麼 L6D6 應該促進生長？

**額外輻射效應**:
- CK: 57 W/m² LED
- L6D6: 57 W/m² LED + 22 W/m² UVA = **79 W/m²**
- 額外輻射 → 額外光合作用 → **促進生長**

**Sun 原始模型驗證**:
- I = 57 W/m² → FW = 86.7g
- I = 79 W/m² → FW = **90.6g** (+4.6%)
- 實驗觀察: FW = 91.4g (+5.1%)
- **Sun 模型完美解釋 UVA 促進效應！**
